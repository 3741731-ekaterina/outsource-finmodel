# Как запустить сервис рассылок

## Первый запуск (один раз)

Откройте Терминал, перейдите в папку проекта и выполните:

```bash
cd /Users/ekaterinaahontova/Documents/Project/prilojenie-rassylok
npm run install:all
```

## Каждый раз

В одном окне Терминала:
```bash
cd /Users/ekaterinaahontova/Documents/Project/prilojenie-rassylok
npm run dev
```

Откроется два процесса:
- Сервер: http://localhost:3001
- Интерфейс: http://localhost:5173 ← открывайте это в браузере

## Настройка BotHelp

1. Войдите в кабинет BotHelp
2. Перейдите: Настройки → Интеграции → REST API
3. Получите Client ID и Client Secret
4. Откройте http://localhost:5173 → раздел «Настройки»
5. Введите Client ID и Client Secret → Сохранить
6. Найдите свой Subscriber ID (инструкция в разделе Настройки)
7. Введите Subscriber ID → Сохранить
8. Нажмите «Проверить соединение»

## Использование

1. Редактор → написать тему и текст письма
2. Выбрать бота
3. Нажать «Отправить тест» — сообщение придёт вам в Telegram
4. Предпросмотр → проверить как выглядит
